처음으로 개인 블로그를 직접 만들어 보았다.
그동안 브런치, 티스토리, 네이버, 벨로그 등 다양한 블로그 플랫폼을 이용했다. 그러면서 글들은 여러 플랫폼에 분산되고 잊혔다.
결국에는 특정 플랫폼에 종속되지 않은 개인 블로그가 필요하다고 생각해 왔다.

그리고 최근 AI 에이전트의 발전으로 개인 블로그 만들기가 놀랍도록 쉬워졌다. 블로그 관리도 AI를 통한 자연어로 가능해졌다.

기존 블로그 플랫폼을 이용하거나, 워드프레스를 이용하는 것에 비해 초기 기능은 적을 수 있다.
하지만 앞으로 직접 원하는 기능을 추가해 나가는 재미가 더 클 것으로 생각한다.

이 블로그는 다음과 같은 기술로 만들어졌다:
- **Next.js 16** — App Router 기반 React 프레임워크
- **MDX** — Markdown + JSX로 글 작성
- **Tailwind CSS v4** — 유틸리티 퍼스트 CSS
- **rehype-pretty-code + shiki** — 코드 블록 하이라이팅
- **next-themes** — 다크모드
- **Giscus** — GitHub Discussions 기반 댓글

블로그가 조금 더 개인적인 공간이 된 만큼, 부담 없이 글을 올리기 시작할 수 있을 것 같다.
